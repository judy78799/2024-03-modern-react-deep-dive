### 1.2.2 함수를 정의하는 4가지 방법

- 함수 선언문

```jsx
function add(a, b) {
	return a + b;
}
```

- 함수 표현식

```jsx
const sum = function(a, b) {
	return a + b;
}
```

둘다 호이스팅이 되지만 함수 표현식은 호이스팅되는 시점에 `var`의 경우에는 `undefined`로 초기화한다는 차이가 있다.

- Function 생성자
- 화살표 함수
    - constructor 사용 불가 ⇒ 생성자 불가
    - argunemts 존재하지 않음
    - `this` 스코프 다름 ⇒ 상위 스코프의 `this`

```jsx
const add = (a, b) => {
	return a + b
}
```
